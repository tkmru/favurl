favurl option

Removing URL option

URL of movie service

You can switch ON(OFF) removing this URL(youtube, vimeo, instagram etc).

URL of picture service

You can switch ON(OFF) removing this URL(twitpic, instagram etc).

URL of tweet

You can switch ON(OFF) removing this URL(twitter.com/username/status/...).

URL of location service

You can switch ON(OFF) removing this URL(foursquare).

Auto open new URL option

You can switch ON(OFF) auto open new favorite url when chrome start.
Clicking the button in the upper right corner of the list screen is also opening them.



Display URL option

You can choose whether to display brief URL as the official client, and whether to display original length.



Language option

You can select English or Japanese.

favurl 詳細設定

削除URL設定

動画サイトのURL

動画サイトのURLを削除する機能のON、OFFを切り替えられます。(youtube, niconico, instagram etc)

写真サービスのURL

写真サービスのURLを削除する機能のON、OFFを切り替えられます。(twitpic, instagram etc)

ツイートのURL

ツイートのURLを削除する機能のON、OFFを切り替えられます。(twitter.com/username/status/...)

位置情報サービスのURL

位置情報サービスのURLを削除する機能のON、OFFを切り替えられます。(foursquare)

オートオープンURL設定

ブラウザを開いた時に、閉じていた間にふぁぼったURLを自動で開く機能のON、OFFを切り替えられます。
OFFにした場合は、ふぁぼ一覧画面の右上のボタンをクリックすることで開けます。



URL表示設定

URLを公式クライアントのように短く表示するか、本来の長さで表示するかを設定できます。



言語設定

日本語または英語を選択できます。
