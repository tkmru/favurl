favurl option

Removing URL

URL of movie service

display/non-display URL like youtube, vimeo, instagram etc.

URL of picture service

display/non-display URL like twitpic, instagram etc.

URL of tweet

display/non-display URL like 'twitter.com/username/status/...'.

URL of location service

display/non-display URL like foursquare, swarm.

Auto open new URL

Auto opening new favorite url when chrome start.
Clicking the button in the upper right corner of the list screen is also opening them.


URL length

display brief URL as the official client or original length.


Notification

In case favurl fail to get favrites and get new favorite URL, it send message.

Language

favurl 詳細設定

削除URL

動画サイトのURL

動画サイトのURLを削除する機能のON、OFFを切り替え。(youtube, niconico, instagram etc)

写真サービスのURL

写真サービスのURLを削除する機能のON、OFFを切り替え。(twitpic, instagram etc)

ツイートのURL

ツイートのURLを削除する機能のON、OFFを切り替え。(twitter.com/username/status/...)

位置情報サービスのURL

位置情報サービスのURLを削除する機能のON、OFFを切り替え。(foursquare, swarm)

オートオープンfavurl

ブラウザを開いた時に、閉じていた間にふぁぼったURLを自動で開く機能のON、OFFを切り替え。
OFFのときは、ふぁぼ一覧画面の右上のボタンをクリックすることで開ける。


URLの長さ

URLを公式クライアントのように短く表示するか、本来の長さで表示するかを切り替え。


通知

ふぁぼを取得出来ない場合や、新しくふぁぼったURLがあった場合の通知のON、OFFを切り替え。


言語

日本語または英語に切り替え。
